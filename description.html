Introduction

Line Segment Intersection Problem is one of the fundamental problems in the field of Computational Geometry. This problem has numerous real-world applications, including geographical information systems (GIS), computer graphics, robotics, and more. It involves finding all the intersections between a set of line segments in a 2D plane. These line segments could represent various objects such as roads, wires, or any linear features, and understanding where they intersect is crucial for solving a wide range of problems efficiently.

Problem

Given a set of line segments on a 2D plane, find all the locations where these line segments cross each other.

Applications

  • To optimize route planning, find all the locations where roads intersect.
  • Alternatively, Find all the possible locations of bridges, by checking the intersections between roads and rivers. (basically spatial analysis)
  • Used in computer graphics for tasks such as collision detection, ray tracing, and determining whether geometric shapes intersect.
  • Used in physics simulations for particle interactions and collision detection in simulations of physical systems.
  • And much more...

Algorithm

function findIntersections(segments):
    events = InitializeEventsList(segments) // Create a list of events (start and end points of segments)
    status = InitializeStatusStructure()    // Create a balanced binary search tree (e.g., AVL tree)

    SortEventsByXCoordinate(events)         // Sort events by their x-coordinates

    intersections = []                      // Initialize an empty list to store intersections

    for each event in events:
        if event.type is 'start':
            status.Insert(event.segment)        // Insert the segment into the status structure
            above = status.Above(event.segment) // Get the segment directly above the current segment
            below = status.Below(event.segment) // Get the segment directly below the current segment

            if above is not null and Intersects(event.segment, above):
                intersections.Append((event.segment, above)) // Store the intersection

            if below is not null and Intersects(event.segment, below):
                intersections.Append((event.segment, below)) // Store the intersection

        else if event.type is 'end':
            above = status.Above(event.segment) // Get the segment directly above the current segment
            below = status.Below(event.segment) // Get the segment directly below the current segment

            if above is not null and below is not null and Intersects(above, below):
                intersections.Append((above, below)) // Store the intersection

            status.Remove(event.segment) // Remove the segment from the status structure

    return intersections
